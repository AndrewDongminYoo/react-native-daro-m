# react-native-daro-m

DaroM React Native Module for Android and iOS

## Installation

```sh
yarn install
```

## Usage


```js
import { DaroMView } from "react-native-daro-m";

// ...

<DaroMView color="tomato" />
```

## Example

```sh
cd example
```

### Android

```sh
yarn run android
```

### iOS

```sh
yarn run ios --simulator "iPhone 16"
```
